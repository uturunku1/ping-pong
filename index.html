Ping-Pong

Rules for Ping-Pong:

Enter any number and our very fancy algorithm will count your number one by one, except...

  • When a number can be divided by 3. In which case it will say the word "Ping"!
  • When a number can be divided by 5. In which case it will say the word "Pong"!
  • When a number can be divided by 3 and 5. In which case it will say "Ping-pong"!

Isn't this fun?
